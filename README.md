## 📁 Git commit format
* Add
Use when add new feature
* Update
Use when update feature
* Fix
Use when fix some bug/error
* Remove
Use when remove some feature
#### Example :
    Add: Home page
    Fix: Layout not responsive
    Remove: Shadow on item list
